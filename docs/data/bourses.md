### 💡 Conseils pour tes Seeders Django :

1.  **Slugification** : Utilise les noms anglais pour générer tes slugs (ex: `civil-engineering`).
2.  **Traduction** : Pour ton modèle `ScholarshipField`, tu as maintenant la base anglaise parfaite. Tu peux utiliser une IA pour traduire les colonnes "Focus" et "Career" en français pour avoir un contenu bilingue riche.
3.  **Limitations** : Note bien les mentions en rouge sur les images (Psychology, Law, Social Work) qui sont marquées comme "Very limited now in China". Tu pourrais ajouter un champ `is_limited` ou une note spécifique dans ta base de données pour ces bourses.
